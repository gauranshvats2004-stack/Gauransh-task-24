📦 Animation & UI Feedback (Tailwind CSS Project)
📌 Overview

This project is a UI showcase built using HTML + Tailwind CSS, demonstrating different types of modern web animations and interactive effects such as hover transitions, loading indicators, notification badges, and staggered animations.

It is designed as a front-end practice project to explore Tailwind’s utility-first animation system.

🚀 Features
🎞 Built-in Animations
animate-spin → Loading spinner
animate-ping → Notification pulse effect
animate-pulse → Skeleton loading effect
animate-bounce → Attention/scroll indicator




🔘 Button Interactions
Scale on hover
Glow / shadow effects
Color transitions
Rotation effects
Lift (translate Y) effect
Combined multi-effect buttons



🧩 Card Animations
Hover lift + shadow
Scale + rotate effects
Border reveal animation
Group-hover interactions



🔔 Notification Badges
Animated ping dot
Pulsing counters
Bouncing notification indicators





⏳ Loading States
Skeleton loaders
Spinner inside buttons
Animated loading dots






⚡ Staggered Animations
Delay-based UI transitions
Sequential hover effects
